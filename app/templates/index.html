3D Visualization System

Previous Next

3DVisSys, or 3D Visualization System, is free open-source software for examining NOAA MRMS precipitation and reflectivity data with 3D interactive visualizations directly in the browser. It features this web application to load and interact with visualizations, as well as backend data pipelines to create those visualizations. It was primarily built with Python 3.9, Flask and Plotly as part of a CISESS research assistantship.

Instructions
Users can use the site's primary functionality on the left selection menu by either selecting a date to graph or simply grabbing the latest graphs available. Graphs are created hourly, but note that the system will automatically skip times that are found to lack precipitation.
To find the latest graphs:
  1. Use the "Get Latest Graphs" button.
To find precipitation and reflectivity graphs by it's hour and date:
  1. Use the calendar to select a date. Dates with available graphs are automatically highlighted and others are unselectable.
  2. Use the clock button under the calendar to select a time. The system matches your selected time with closest hour collected.
  3. Use the "Get Graphs from Selected Date" button.
